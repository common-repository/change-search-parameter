Replace or Change Default Search Query Parameter With Your Custom Parameter.

== Description ==

This plugin replaces or changes the defualt inbuilt search query paramenter in WordPress with your custom search query parameter. 

Visit <a href="http://www.priteshgupta.com/plugins/change-search-parameter" target="_blank">Change Search Parameter Plugin Homepage</a> for more information.

<strong><a href="http://www.priteshgupta.com/contact" target="_blank">Contact Me</a> if you will like to see any other feature on this plugin.</strong>

== Installation ==

1. Download the latest version.
2. Extract it in the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Customize the plugin in the Settings > Change Search Parameter. 

Visit <a href="http://www.priteshgupta.com/plugins/change-search-parameter" target="_blank">Change Search Parameter Plugin Homepage</a> for more information.
